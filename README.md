# Sam's C/C++ Project MaKer
## Generate a simple C/C++ project with a CMake configuration.
### Usage: cpmk \<language\> \<project-name\>

#### Example C:
```sh
cpmk c c-project
```

#### Example C++:
```sh
cpmk cpp cpp-project
```
or
```sh
cpmk cc cpp-project
```

# Install
### Requires Cargo/Rust
Simply run
```sh
cargo install cpmk
```

# Uninstall
Simply run
```sh
cargo uninstall cpmk
```
